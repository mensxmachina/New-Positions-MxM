# Expression of Interest for IT/DevOps Position (EIC Transition Project)

## Role Overview
We are building a Python-based, agentic data analytics platform. We are looking for a Junior IT/DevOps Engineer to assist in infrastructure automation, deployment, and operational support.

You will work closely with the Senior Infrastructure Engineer to maintain a stable, secure, and scalable environment while gaining hands-on experience in cloud operations and CI/CD pipelines.

## Key Responsibilities
- Manage equipment, server configurations, updates, and backups 
- Support website development and monitoring
- Assist in CI/CD pipeline setup and maintenance 
- Support cloud infrastructure management (AWS/Azure, etc) 
- Help with containerization and orchestration (Docker, Kubernetes) 
- Monitor and troubleshoot system performance, logging, and alerts 
- Assist in automation of deployment and operational tasks 
- Maintain documentation of infrastructure processes and procedures 

## Required Skills
- Basic experience with Python scripting 
- Familiarity with Linux/Windows server administration 
- Understanding of CI/CD concepts 
- Basic knowledge of cloud platforms (AWS, Azure, etc) 
- Willingness to learn and follow best practices in DevOps and IT 
- Good troubleshooting and problem-solving skills 

## Nice to Have
- Exposure to website development
- Basic knowledge of networking, firewalls, and security 
- Docker and containerization concepts 
- Familiarity with Kubernetes or other orchestration tools 
- Familiarity with infrastructure as code tools (Terraform, CloudFormation) 
- Experience with monitoring and logging tools (Prometheus, Grafana, ELK stack) 

## What Success Looks Like
- Smooth deployment and operation of test and production environments 
- Reliable CI/CD pipelines supporting all engineering teams 
- Minimal downtime and proactive resolution of infrastructure issues 
- Effective support for Senior Infrastructure Engineer and other teams 
- Increasing responsibility and autonomy over time
