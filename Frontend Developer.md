## Expression of Interest for Frontend Developer Position (EIC Transition Project)

## Role Overview
We are building an agentic data analytics platform where users interact with intelligent systems.

We are looking for a Frontend Engineer (preferably Angular) to design and implement intuitive interfaces.

## Key Responsibilities
- Design scalable frontend architecture  
- Build dashboards and visualizations  
- Develop UI for agent interaction  
- Translate designs into production UI  
- Integrate APIs and real-time data  
- Ensure responsive design  
- Implement testing strategies  
- Optimize performance  

## Required Skills
- 3+ years frontend experience  
- Strong TypeScript, HTML, CSS  
- Experience with visualization libraries  
- Experience with dashboards  
- UI/UX knowledge  
- API integration  

## Nice to Have
- WebSockets / real-time data  
- AI/ML UI experience  
- Accessibility knowledge  
- Design systems  

## What Success Looks Like
- Intuitive UI  
- Smooth user-agent interaction  
- High performance  
- Consistent design  
- Alignment with design prototypes
