# Expression of Interest for Frontend Developer Position (EIC Transition Project)

## Role Overview
We are building an agentic data analytics platform where users interact with intelligent systems that orchestrate data processing and machine learning workflows.

We are looking for a Frontend Engineer with strong, preferably, Angular expertise and UI/UX sensibility to design and implement the user-facing layer of the platform, turning complex analytics and agent behavior into intuitive, responsive interfaces.

## Key Responsibilities
- Design and develop scalable frontend architecture  
- Build interactive dashboards and data visualizations for analytics workflows 
- Develop UI components for agent interaction (chat, workflows, task execution views) and data exploration and results interpretation 
- Translate Figma/Sketch/Zeplin designs into production-ready UI 
- Collaborate with backend and AI teams to integrate APIs and real-time data 
- Ensure responsive design and cross-browser/device compatibility 
- Implement frontend testing strategies (unit, integration, etc.) 
- Optimize performance (lazy loading, efficient state management, rendering optimization) 

## Required Skills
- Strong experience  (3+ years) with developing user interfaces and data visualization libraries (D3.js, Chart.js, etc.) in web development 
- Solid knowledge of TypeScript, HTML, CSS 
- Experience building data-heavy or dashboard-style applications 
- Familiarity with UI/UX principles and design systems 
- Experience working with design tools (Figma, Sketch, or Zeplin) 
- Experience integrating REST/GraphQL APIs 

## Nice to Have
- Familiarity with real-time interfaces (WebSockets, streaming data) 
- Experience designing interfaces for AI/ML systems or chat-based UIs 
- Basic understanding of accessibility and usability standards 
- Exposure to design systems and component libraries 

## What Success Looks Like
- Clean, intuitive UI that simplifies complex analytics workflows 
- Seamless interaction between user, agent, and system outputs 
- High-performance frontend even with large datasets 
- Consistent design language across the platform 
- Tight alignment between design prototypes and final implementation
